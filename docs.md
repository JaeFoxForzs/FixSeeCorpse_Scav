# 📋 Scav Prototype — Shrapnel System Mod
## Game Design Document v1.0

---

## 1. Общее описание

**Название мода:** Shrapnel Overhaul
**Цель:** Расширить систему взрывов, добавив реалистичную механику разлёта осколков при взрыве и остаточного debris на земле после взрыва. Сделать взрывы более опасными, тактически глубокими и атмосферными.

**Философия дизайна:**
- Взрыв — это не просто "бум и урон", а событие с **долгосрочными последствиями**
- Игрок всегда должен иметь **инструменты контроля** (видимость, обувь, осторожность)
- Механика должна **органично вписываться** в существующие системы игры (shrapnel wounds, glass shards, медицина)

---

## 2. Источники взрывов

### 2.1 Противопехотная мина (Landmine)

| Параметр | Значение |
|----------|----------|
| Радиус взрыва | 12 тайлов |
| Базовый урон | До 1000 |
| Тип шрапнеля | Металлический (острый, зазубренный) |
| Кол-во осколков | 20–30 штук |
| Конус разлёта | 360° (равномерно) |
| Дальность полёта осколков | 8–14 тайлов |
| Шанс шрапнеля при попадании | 40–60% на конечность |
| Penetration | Высокий |
| Остаточный debris | Да, 8–9 тайлов от эпицентра |

**Особенности:**
- Самый опасный источник шрапнеля в игре
- Осколки преимущественно поражают **нижнюю часть тела** (ноги, таз, нижний торс)
- Высокий шанс **внутреннего кровотечения** и **переломов**
- Debris держится дольше всех (тяжёлый металл)

---

### 2.2 Динамит (Dynamite)

| Параметр | Значение |
|----------|----------|
| Радиус взрыва | 10 тайлов |
| Базовый урон | Средне-высокий |
| Тип шрапнеля | Смешанный (камень + дерево + лёгкий металл) |
| Кол-во осколков | 12–20 штук |
| Конус разлёта | 360° (хаотично, неравномерно) |
| Дальность полёта осколков | 5–10 тайлов |
| Шанс шрапнеля при попадании | 25–40% на конечность |
| Penetration | Низкий–средний |
| Остаточный debris | Да, 5–7 тайлов от эпицентра |

**Особенности:**
- Хаотичный разлёт — непредсказуемые направления
- Много **поверхностных ран**, меньше глубоких
- Осколки **легче вытаскивать** (дерево/камень менее зазубрены)
- При взрыве в **замкнутом пространстве** (пещера, коридор) — бонус +50% к количеству осколков (рикошет от стен)
- Debris быстрее исчезает (лёгкий материал)

---

### 2.3 Турель (Turret Explosion)

| Параметр | Значение |
|----------|----------|
| Радиус взрыва | 9 тайлов |
| Базовый урон | 0–400 |
| Тип шрапнеля | Тяжёлый металл (болты, шестерёнки, платы, корпус) |
| Кол-во осколков | 6–12 штук |
| Конус разлёта | 360° (неравномерно, больше вверх и в стороны) |
| Дальность полёта осколков | 6–11 тайлов |
| Шанс шрапнеля при попадании | 30–50% на конечность |
| Penetration | Очень высокий |
| Остаточный debris | Да, 6–8 тайлов от эпицентра |

**Особенности:**
- **Мало осколков, но каждый — серьёзный** (тяжёлые куски металла)
- Высокий шанс **перелома** при попадании
- Уникальный эффект: **Electronic Shrapnel** — 15% шанс на осколок с остатками электроники
  - Вызывает кратковременный **stun** (0.5–1 сек) при попадании
  - При вытаскивании — шанс на дополнительный урон (короткое замыкание)
- Debris содержит **scrap metal** — можно собирать как ресурс

## 3. Типы шрапнеля

### 3.1 Таблица типов

| Тип | Источник | Урон за осколок | Bleed | Penetration | Сложность извлечения | Шанс инфекции | Особый эффект |
|-----|----------|-----------------|-------|-------------|----------------------|---------------|---------------|
| **Металлический (острый)** | Мина | 8–15 | Сильный | Высокий | Сложно (зазубрины) | 20% | Внутреннее кровотечение |
| **Каменный** | Динамит / стены | 4–8 | Средний | Средний | Легко | 35% (грязь) | — |
| **Деревянный** | Динамит / ящики | 3–6 | Слабый | Низкий | Легко | 40% (занозы) | — |
| **Тяжёлый металл** | Турель | 12–20 | Средний | Очень высокий | Сложно (большой) | 15% | Перелом (25%) |
| **Electronic** | Турель | 5–10 | Слабый | Средний | Средне | 10% | Stun 0.5–1 сек |
| **Стеклянный** | Окружение (лампы, бутылки) | 3–7 | Сильный | Низкий | Средне (хрупкий, может сломаться внутри) | 30% | При неудачном извлечении — доп. урон |

### 3.2 Вторичный шрапнель (от окружения)

Взрыв может разрушать объекты вокруг, которые порождают **собственные осколки**:

| Разрушенный объект | Тип вторичного шрапнеля | Кол-во | Радиус разлёта |
|--------------------|-------------------------|--------|----------------|
| Каменная стена | Каменный | 3–6 | 3–5 тайлов |
| Деревянный ящик | Деревянный | 2–5 | 2–4 тайла |
| Лампа / бутылка | Стеклянный | 4–8 | 2–3 тайла |
| Металлическая бочка | Металлический | 3–5 | 4–6 тайлов |

## 4. Остаточный Debris (наземные осколки)

### 4.1 Спавн

После любого взрыва на земле появляются **кластеры debris**:

| Параметр | Мина | Динамит | Турель |
|----------|------|---------|--------|
| Радиус зоны debris | 8–9 тайлов | 5–7 тайлов | 6–8 тайлов |
| Покрытие площади | 35–45% | 20–30% | 25–35% |
| Паттерн | Кольцо вокруг эпицентра (центр выжжен) | Хаотичные пятна | Неравномерные скопления |

### 4.2 Взаимодействие при наступлении

| Параметр | Значение |
|----------|----------|
| Шанс получить шрапнель | 55–75% (без обуви) |
| Кол-во осколков за шаг | 1–4 (в ноги: ступни, голени) |
| Модификатор: Reinforced Boots | Шанс снижен до 10–20% |
| Модификатор: Обычная обувь | Шанс снижен до 35–50% |
| Модификатор: Босиком | Базовый шанс + доп. урон |
| Модификатор: Slow Walk / Crouch | Шанс снижен на 40–50% |
| Модификатор: Бег / Sprint | Шанс повышен на 15–20%, доп. penetration |

### 4.3 Время жизни Debris

| Тип debris | Время жизни | Обоснование |
|------------|-------------|-------------|
| Металлический (мина) | 15–20 минут | Тяжёлый, никуда не денется |
| Смешанный (динамит) | 8–12 минут | Лёгкий материал, разносит ветром |
| Тяжёлый металл (турель) | 18–25 минут | Массивные куски |
| Стеклянный (окружение) | 10–15 минут | Хрупкий, но острый |

### 4.4 Видимость и обнаружение

| Условие | Видимость |
|---------|-----------|
| Дневной свет / яркий фонарь | Хорошо видно — блестящие точки на земле |
| Тусклый свет | Средне видно — нужно присмотреться |
| Полная темнота | Почти не видно — только звук хруста при приближении (1–2 тайла) |
| С фонарём близко | Видно отчётливо — блики от металла |

**Аудио-подсказки:**
- При приближении к debris (2 тайла) — тихий звук хруста/скрежета под ногами
- При наступлении — резкий звук + pain grunt персонажа

### 4.5 Контрмеры и взаимодействие

| Действие | Эффект | Время |
|----------|--------|-------|
| **Расчистка лопатой (Shovel)** | Полностью убирает debris с 1 тайла | 2–3 сек на тайл |
| **Расчистка руками** | Убирает debris, но 30% шанс порезаться | 4–5 сек на тайл |
| **Сбор осколков (Tweezers/руки)** | Собирает металл как ресурс для крафта | 3–4 сек на тайл |
| **Обход** | Просто обойти зону | — |
| **Бросок предмета** | Бросить что-то на debris — визуально показывает опасную зону | — |

## 5. Медицинские последствия

### 5.1 Шрапнель в теле

Использует и расширяет **существующую систему** Scav Prototype:

| Эффект | Описание |
|--------|----------|
| **Постоянный Bleed** | Пока осколок внутри — кровотечение не останавливается полностью |
| **Боль** | +Pain пока осколок в теле, масштабируется от типа и глубины |
| **Дебафф конечности** | Нога → хромота, скорость −20–40%. Рука → точность −15–30%. Торс → stamina −20% |
| **Блок лечения** | Нельзя полностью залечить рану, пока осколок не извлечён |
| **Инфекция** | Шанс инфекции растёт со временем, если осколок не извлечён (зависит от типа шрапнеля) |

### 5.2 Извлечение

| Параметр | Лёгкий осколок (дерево/камень) | Средний (стекло/лёгкий металл) | Тяжёлый (металл мины/турели) |
|----------|-------------------------------|-------------------------------|------------------------------|
| Инструмент | Руки / Tweezers | Tweezers | Tweezers обязательно |
| Время | 2–4 сек | 4–7 сек | 6–10 сек |
| Боль при извлечении | Низкая | Средняя | Высокая |
| Шанс облома (осколок ломается внутри) | 5% | 15% (стекло: 25%) | 10% |
| Доп. урон при облоге | 2–4 | 4–8 | 6–10 |

## 6. Крафт из собранных осколков

Собранные с земли или извлечённые из тела осколки можно использовать:

| Рецепт | Ингредиенты | Результат |
|--------|-------------|-----------|
| **Импровизированная мина** | 8x металлический шрапнель + Gunpowder + Wire | Слабая мина (50% эффективности обычной) |
| **Shrapnel-патроны** | 4x любой шрапнель + Casing + Gunpowder | Патроны с +15% spread, −10% дальность, шанс шрапнеля при попадании |
| **Caltrops (ловушка)** | 6x металлический шрапнель | Бросаемая зона debris 3x3 тайла |
| **Импровизированный щит-вставка** | 10x тяжёлый металл | Вставка в броню, +10% защита от шрапнеля |

## 7. Настройки мода (Config)

Для гибкости — все ключевые параметры выносим в конфиг:

```ini
[General]
EnableShrapnelOverhaul = true
EnableDebris = true
EnableSecondaryShranel = true
EnableCrafting = true

[Shrapnel]
GlobalDamageMultiplier = 1.0
GlobalShrapnelChance = 1.0
MaxShrapnelPerExplosion = 30
MaxDebrisLifetimeMinutes = 20

[Debris]
DebrisCoveragePercent = 0.4
StepDamageChance = 0.65
BootProtectionMultiplier = 0.3
CrouchProtectionMultiplier = 0.5
SprintPenaltyMultiplier = 1.2

[Visuals]
DebrisParticleIntensity = 1.0
ShrapnelTrailEnabled = true
DebrisGlintInDarkness = false

[Medical]
InfectionChancePerMinute = 0.02
ShrapnelPainMultiplier = 1.0
ExtractionFailChance = 0.1
```
